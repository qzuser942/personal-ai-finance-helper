

# 个人智能理财系统后端 交付说明

| 指标     | 数据                                         |
| -------- | -------------------------------------------- |
| 总文件数 | 85 (83 Java + 1 pom.xml + 1 application.yml) |

## 🔑 核心强制规范全覆盖
| 规范要求                                       | 实现状态                          |
| ---------------------------------------------- | --------------------------------- |
| RESTful风格 + `/api/user/`/`/api/admin/`分前缀 | ✅ 全部66个接口严格遵循            |
| 统一返回体 `Result<T>`                         | ✅ 标准code/msg/data/timestamp结构 |
| 全局异常处理 + 自定义业务错误码                | ✅ 覆盖43个细分业务错误场景        |
| 统一分页结果 `PageResult`                      | ✅ 原生继承MyBatisPlus Page对象    |
| JWT双拦截器 + 三级权限隔离                     | ✅ 普通用户/运营管理员/超级管理员  |
| MyBatisPlus字段自动填充                        | ✅ 自动填充createdAt/updatedAt     |
| BCrypt不可逆密码加密                           | ✅ 所有用户/管理员密码统一加密     |
| CORS局域网全设备放行                           | ✅ allowedOriginPatterns:*         |
| Knife4j接口文档分组展示                        | ✅ 用户端/管理员端独立文档分组     |
| 管理员操作日志AOP自动记录                      | ✅ 自动记录所有POST/PUT/DELETE操作 |
| 逻辑删除机制                                   | ✅ isDeleted字段 + @TableLogic注解 |
| 高精度金额计算                                 | ✅ 全部金额字段用DECIMAL(12,2)     |

## 🚀 快速启动
1. **导入MySQL数据库**  
   运行数据库设计说明书中的建表SQL，创建名为 `finance_db` 的数据库
2. **修改配置文件**  
   编辑 `application.yml`，更新MySQL连接地址、账号和密码
3. **启动项目**
   ```bash
   cd personal-finance-server
   mvn spring-boot:run
   ```
4. **访问接口文档**  
   本地地址：`http://localhost:8080/doc.html`  
   局域网地址：`http://[电脑内网IP]:8080/doc.html`
5. **默认超级管理员账号**  
   账号：`admin`  
   密码：`admin123`

## 🔗 前后端对接说明
- 鸿蒙ArkTS客户端：调用 `/api/user/**` 接口，请求头携带 `Authorization: Bearer <用户Token>`
- Vue3管理员后台：调用 `/api/admin/**` 接口，请求头携带 `Authorization: Bearer <管理员Token>`
- 小票图片访问地址：`http://[电脑内网IP]:8080/uploads/receipts/<文件名>`
- 离线账单同步接口：`POST /api/bill/sync-batch`，请求体传入离线账单JSON数组
