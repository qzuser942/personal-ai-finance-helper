## 三、🚀 项目启动步骤
```bash
# 进入项目目录
cd finance-admin-web

# 安装全部依赖
npm install

# 按需修改 vite.config.js 内代理 target 后端IP地址

# 启动开发服务
npm run dev
```
1. 浏览器访问地址：`http://localhost:3000`
2. 默认管理员账号：账号 `admin` / 密码 `admin123`
