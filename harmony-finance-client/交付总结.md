## 三、🔧 项目本地运行配置说明
1. 打开文件路径：`entry/src/main/ets/store/AppState.ets`
2. 修改第23行局域网后端IP地址：
```typescript
static BASE_URL: string = 'http://你的电脑局域网IP:8080';
```
3. 完整运行流程：
导入项目至DevEco Studio → 同步工程依赖 → 启动后端服务 → 模拟器/真机运行项目即可前后端联调
